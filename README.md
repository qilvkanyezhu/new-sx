### 3项目功能
- 用户：注册、登录、删除、修改、查询
- 商品：查询、删除、添加
- 购物车：添加、删除商品至购物车、购物车列表、修改购物车条目中的购买数量、修改购物车条目中的是否勾选、购物车列表（勾选商品）
